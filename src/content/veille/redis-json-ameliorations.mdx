# Résumé

Redis a ajouté les commandes JSON.MERGE et JSON.MSET dans la version 7.2, optimisant la manipulation des données JSON complexes et structurées directement dans la base. Ces nouvelles commandes simplifient les opérations de fusion et de mise à jour multiple de documents JSON, réduisant le nombre d'opérations réseau nécessaires et améliorant les performances globales.
