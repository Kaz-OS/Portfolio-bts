# Résumé

Étude comparant les performances des systèmes de base de données MongoDB (Document) et Redis (Clé-Valeur) dans des applications basées sur les microservices. Cette analyse aide les architectes à choisir la solution la plus adaptée selon les besoins spécifiques : MongoDB pour la persistance et les requêtes complexes, Redis pour le cache et les performances ultra-rapides.
