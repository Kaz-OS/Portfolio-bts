# Résumé

Redis 7.2 intègre la capacité d'effectuer des requêtes géospatiales sophistiquées par recherche par polygone, très utilisée dans les applications de logistique et de cartographie. Cette fonctionnalité permet de définir des zones géographiques complexes et d'effectuer des recherches de points à l'intérieur de ces polygones, ouvrant de nouvelles possibilités pour les applications de géolocalisation.
